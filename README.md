# DispMov-TP6-MensajesServicio

Crear una aplicación que cuando se ejecute su actividad principal, se inicie un servicio que cada 9000 milisegundos 
acceda al Content Provider de los mensajes y muestre por la consola de debug (Log.d) los datos más 
importantes de los últimos 5(cinco) mensajes recibidos. 
La vista asociada a la Activity Principal poseerá 2 botones, uno para iniciar el servicio y otro para finalizarlo
